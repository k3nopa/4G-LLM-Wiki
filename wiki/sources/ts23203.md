# 3GPP TS 23.203 v18.0.0 — Policy and Charging Control Architecture

**Full title**: Policy and charging control architecture (Release 18, 2024-04)
**PDF**: `raw/papers/ts_123203v180000p.pdf`
**Pages**: ~271 spec pages

## Scope

Specifies stage-2 Policy and Charging Control (PCC) for IP-CANs including:
- Flow Based Charging (FBC) — credit control, offline charging for SDFs
- Policy control — gating, QoS, redirection

Covers: E-UTRAN/LTE (3GPP), GPRS (GERAN/UTRAN), non-3GPP accesses (via TS 23.402), Fixed Broadband (Annex P/S), DOCSIS, WiMAX (informative).

### Annex A.4 + A.5 + Annex H — EPC Access-Specific PCC

**Annex A.4 (3GPP GTP-based EPC):** PCEF at PDN-GW, no BBERF, Case 1. PCEF performs bearer binding. Bind to Default Bearer supported. Default EPS Bearer QoS (QCI+ARP) authorized by PCRF via Gx; simultaneous modification of all default-QoS-matching PCC rules on change. EPS credit re-auth triggers (Table A.4.3-1): SGSN change, SGW change, RAT change, location changes (TA/RA/ECGI/CGI-SAI/eNodeB), PRA presence, User CSG info change. EPS event triggers (Table A.4.3-2): same plus Subscribed APN-AMBR change, EPS Subscribed QoS change, 3GPP PS Data Off.

**Annex A.5 (3GPP PMIP-based EPC):** PCEF at PDN-GW, BBERF at SGW (Gxc). BBERF performs bearer binding and enforces Default EPS Bearer QoS. Same event triggers at BBERF. PCRF modifies all QoS rules matching default bearer QoS on Default EPS Bearer QoS change.

**Annex H (EPC-based non-3GPP access):**
- **H.2 cdma2000 HRPD**: BBERF in HSGW (Gxa). HSGW supports all Gxa procedures. HSS can signal BBERF to skip Gateway Control Session establishment. During EUTRAN-to-HRPD pre-registration, HSGW is non-primary BBERF associated with UE IP-CAN sessions. PCRF provides UE IPv6 prefix to HSGW during GCS establishment.
- **H.3 Trusted WLAN S2a**: No BBERF; Case 1. PCEF provides TWAN location info (TWAN ID/Time Zone), IP-CAN type, trusted-access indication. Only trigger: RAT type change. Access Network Info Report applies. TWAN Release Cause provided at termination.
- **H.4 Untrusted non-3GPP (ePDG/S2b)**: No BBERF; Case 1. PCEF provides ePDG IP address, Serving Network, untrusted indication. Location info includes ePDG IP (IKEv2) + UE local IP/port (if NAT). PCRF reports ePDG IP to AF (P-CSCF); UE local IP not reliable (spoofable). UWAN Release Cause at termination.

**Annex G (informative):** PCC rule precedence ordering for overlapping SDFs: dynamic PCC rules > predefined rules known to PCRF > predefined rules not known to PCRF > dynamic non-operator-controlled rules.
