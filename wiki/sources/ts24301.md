# 3GPP TS 24.301 v17.6.0 — NAS Protocol for EPS (Stage 3)

**Full title:** Non-Access-Stratum (NAS) protocol for Evolved Packet System (EPS); Stage 3  
**Release:** Release 17 (2022-05)  
**File:** `raw/papers/ts_124301v170600p.pdf`  
**Total length:** ~609 pages

---

## Scope

Stage-3 (protocol) specification for NAS signalling between UE and MME. Defines:
- **EMM** (EPS Mobility Management): registration, attach, detach, TAU, service request, security
- **ESM** (EPS Session Management): bearer activation/modification/deactivation, PDN connectivity
- Message formats and IE encoding for all NAS messages
- Timer values for both EMM and ESM

Companion to stage-2 (architecture) specs TS 23.401 (EPC) and TS 23.228 (IMS).

## Chunk 4 — §7 + §8 (2026-04-20)

Ingested §7 (error handling: message too short, unknown PTI/EBI, unknown message type, mandatory IE errors, unknown IEIs, non-imperative errors, semantic errors) and §8 (message functional definitions: §8.2 all 34 EMM messages + §8.3 all 25 ESM messages with their IE tables and conditional rules).

**Key content:**

- §7.1–7.8: Error handling precedence — §7.2 too-short → silently ignore; §7.3.1 unknown PTI → ESM STATUS cause #81 or REJECT #47 depending on message; §7.3.2 unknown EBI → STATUS #43 or accept DEACTIVATE; §7.4 unknown message type → STATUS #97/#98; §7.5 mandatory IE error in most ESM REQUESTs → REJECT #96, DEACTIVATE → ACCEPT regardless; §7.6 unknown IEIs → ignore; §7.7 non-imperative errors → treat as absent; §7.8 semantic errors → STATUS #95
- §8.2 EMM messages: 34 messages total, all dual-significance except EMM INFORMATION and EMM STATUS (local). Key structural notes: ATTACH REQUEST has ~45 IEs; TAU REQUEST has ~40 IEs; ATTACH ACCEPT/TAU ACCEPT include all negotiated parameters (T3412/GUTI/TAI list/T3324/Extended DRX/T3447/T3448/ciphering key data). SECURITY MODE COMMAND includes Hash_MME for integrity-absent attach/TAU attack detection. SERVICE REQUEST uses non-standard structure (no standard security header + message type — uses KSI + 5-bit SN + Short MAC). CONTROL PLANE SERVICE REQUEST (NB-S1) can piggyback ESM message container or NAS message container.
- §8.3 ESM messages: 25 messages total. ACTIVATE DEFAULT includes APN-AMBR, Connectivity type (LIPA), Header compression config, Control plane only indication, Serving PLMN rate control. ACTIVATE DEDICATED includes Extended EPS QoS for high-rate bearers. BEARER RESOURCE MODIFICATION REQUEST can carry Header compression configuration for ROHC re-negotiation (CIoT or 5G-to-4G inter-system). ESM DUMMY MESSAGE used during emergency attach without PDN request. REMOTE UE REPORT/RESPONSE for ProSe relay. ESM DATA TRANSPORT with Release assistance indication for NAS release optimization.
- §9.8 message type tables: EMM messages identified by bit pattern 01xxxxxx; ESM messages by 11xxxxxx; specific hex codes documented.
