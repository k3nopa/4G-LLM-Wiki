# 3GPP TS 29.328 v16.1.0 — IMS Sh Interface: Signalling Flows and Message Contents

**Full title:** IP Multimedia (IM) Subsystem Sh interface; Signalling flows and message contents
**Release:** 16 (2020-11)
**Companion spec:** TS 29.329 (Sh Diameter protocol details)

## Coverage

This spec defines the **signalling flows and message contents** for the Sh (AS↔HSS) and Dh (AS↔SLF) interfaces. It is the stage-3 counterpart to the stage-2 architecture in TS 23.228 and TS 23.218.
